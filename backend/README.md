# Backend - Express Server

## Setup

1. Install dependencies:

   ```bash
   npm install
   ```

2. Create a `.env` file (see `.env.example` for reference):

   ```env
   PORT=3000
   ```

3. Start the server:
   ```bash
   node server.js
   ```

## Endpoints

- `GET /` - Returns a welcome message.

## Environment Variables

- `PORT`: Port number for the server (default: 3000)
